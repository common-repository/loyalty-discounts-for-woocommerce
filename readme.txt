== Description ==

Apply WooCommerce loyalty discounts and rewards to a customers checkout, based specific rules and criteria that needs to be met by the user.

Currently there are 3 different types of WooCommerce discounts that you can setup for your loyal customers:

-  Discount based on users lifetime total orders on your website.
-  Discount based on the users current cart total.
-  Discount based on users lifetime total number of items purchased.

The loyalty based WooCommerce discounts can be applied/limited to specific user roles.

You can use shortcode [loyalty-discount-info] on a page to allow users to view their current progress and eligibility to the available loyalty discounts.

Get started today and offer WooCommerce rewards and loyalty discounts to your customers with this simple and easy to use plugin.

== Installation ==

1. Upload 'woo-loyalty-discounts' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customise plugin settings in WordPress admin at Settings > Woo Loyalty Discounts.
